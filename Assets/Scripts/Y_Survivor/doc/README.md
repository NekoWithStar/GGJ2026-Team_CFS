## ✅ 配置检查清单

### 武器配置

```
□ Weapon 数据已创建
□ 武器类型已选择（Melee/Ranged）
□ 基础属性已填写
□ 远程武器：projectilePrefab 已配置
□ 近战武器：meleeRange > 0
□ weaponPrefab 已关联
□ Prefab 包含必需组件（WeaponControl + WeaponPropertyManager）
```

### 属性卡配置

```
□ PropertyCard 数据已创建
□ 稀有度已选择
□ targetType 已设置
□ propertyType 已设置
□ modType 已选择
□ modValue 数值合理
```

## 🔑 核心系统说明

### 武器系统

- 支持远程/近战两种类型
- 可配置多种发射模式（单发、散射、螺旋、环形）
- 自动管理 8 种武器属性
- 支持暴击系统

### 属性卡系统

- 修改 3 种目标（武器/玩家/敌人）
- 支持 3 种修改方式（加法/乘法/后置加法）
- 效果永久累积
- 切换武器时保持所有加成
